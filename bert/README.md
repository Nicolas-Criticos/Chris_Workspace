# Bert's Knowledge Base

This directory stores all information Bert receives from Nicris and Chris.

## Structure

- **incoming/** - Raw submissions, dated and tagged
- **organized/** - Categorized by topic (business, technical, personal, etc.)
- **archives/** - Older materials, searchable
- **index.md** - Master index of all stored knowledge

## Bert's Rules

1. **Store everything** — No deletion without explicit request
2. **Organize as received** — Sort by date, type, and relevance
3. **Keep clean** — Clear naming, consistent format
4. **Chris has full access** — Read, retrieve, query anything
5. **Respond promptly** — Answer both Nicris and Chris queries immediately

## Access

- **Nicris:** Send anything, request anything back
- **Chris:** Query Bert for context, pull full files, request synthesis
- **Storage:** All files versioned and dated
